BCapital Techno ("nosotros", "nuestra app") opera una aplicación interna integrada con la plataforma de Meta (Facebook e Instagram) con el propósito de gestionar y analizar campañas publicitarias propias y de clientes.

02 · Datos que recopilamos

A través de la integración con la API de Meta Ads, nuestra aplicación accede a los siguientes tipos de datos:

  • Métricas de rendimiento de campañas publicitarias (impresiones, clics, conversiones, gasto)
  • Información de cuentas publicitarias vinculadas a nuestra cuenta de Business Manager
  • Datos de páginas de Facebook e Instagram administradas por BCapital Techno
  • Información sobre audiencias personalizadas y píxeles de Meta asociados a nuestras cuentas
  • Datos de configuración de campañas, conjuntos de anuncios y creatividades

Importante: La aplicación no recopila datos personales de usuarios finales de Facebook o Instagram. Solo accedemos a datos de gestión publicitaria dentro de las cuentas autorizadas explícitamente.

03 · Finalidad del tratamiento

Los datos accedidos a través de la API de Meta se utilizan exclusivamente para:

  • Leer y analizar métricas de campañas para optimización del rendimiento
  • Crear, editar y pausar campañas publicitarias en nombre de las cuentas autorizadas
  • Gestionar publicaciones y contenido de páginas vinculadas
  • Administrar audiencias personalizadas y configuración de píxeles para segmentación publicitaria
  • Generar reportes internos de rendimiento de medios pagados

No utilizamos estos datos para ningún propósito diferente al descrito, ni los cedemos a terceros con fines comerciales.

04 · Permisos de la plataforma Meta

Nuestra aplicación solicita los siguientes permisos (scopes) dentro de la plataforma de Meta:

  • ads_read — lectura de campañas, conjuntos de anuncios, anuncios y métricas de rendimiento
  • ads_management — creación y edición de campañas publicitarias
  • pages_manage_ads — gestión de anuncios vinculados a páginas de Facebook
  • pages_read_engagement — lectura de métricas de páginas
  • business_management — acceso a la información del Business Manager

Todos los permisos se utilizan exclusivamente para los fines declarados en esta política y conforme a los Términos de Servicio para Desarrolladores de Meta.

05 · Almacenamiento y seguridad

Los tokens de acceso y credenciales de la API de Meta se almacenan de forma segura y no son compartidos con terceros. Implementamos las siguientes medidas de seguridad:

  • Almacenamiento encriptado de tokens de acceso
  • Acceso restringido al personal autorizado de BCapital Techno
  • No se almacena localmente información sensible de cuentas publicitarias
  • Las conexiones a la API de Meta se realizan exclusivamente a través de HTTPS

06 · Compartición de datos

BCapital Techno no vende, alquila ni comparte los datos accedidos a través de la API de Meta con terceros, excepto en los siguientes casos:

  • Cuando sea requerido por una autoridad legal competente bajo la legislación colombiana vigente
  • Para el funcionamiento técnico de la aplicación con proveedores de infraestructura que actúan como encargados del tratamiento bajo acuerdo de confidencialidad

07 · Retención de datos

Los datos de rendimiento de campañas se conservan únicamente durante el tiempo necesario para cumplir los fines descritos en esta política. Los tokens de acceso de la API de Meta se revocan o eliminan cuando:

  • El usuario desconecta la integración desde la plataforma de Meta
  • Finaliza la relación comercial que justificó el acceso
  • El usuario solicita expresamente la eliminación de sus datos

08 · Derechos del usuario

De conformidad con la Ley 1581 de 2012 (Habeas Data) de Colombia, los usuarios tienen derecho a:

  • Conocer, actualizar y rectificar sus datos personales
  • Solicitar prueba de la autorización otorgada
  • Ser informados sobre el uso de sus datos
  • Revocar la autorización y solicitar la supresión de los datos
  • Acceder gratuitamente a sus datos personales

Para ejercer cualquiera de estos derechos, puede contactarnos en privacidad@bcapitaltechno.com.

09 · Uso de la API de Meta — cumplimiento

Esta aplicación cumple con las Políticas de la Plataforma de Meta y las Condiciones de uso de la API de Marketing de Meta. Los datos obtenidos a través de la API de Meta no se utilizan para fines distintos a los declarados, no se comparten con redes de publicidad de terceros y no se usan para perfilar usuarios finales de las plataformas de Meta.

10 · Cambios en esta política

Podemos actualizar esta Política de Privacidad periódicamente. Cualquier cambio significativo será notificado a través de los canales de comunicación habituales o mediante un aviso en esta misma página. La fecha de última actualización aparece siempre al inicio del documento.
